# java-lotto-precourse

## 기능
1. 로또 구입 금액 입력받기 -> 입력값 확인
2. 금액 만큼 로또 발행하기(클래스 이용)
3. 당첨번호 입력받기(쉼표를 기준으로 구분) -> 입력 값 확인 & 중복 검사
4. 보너스 번호 입력받기 -> 입력 값 확인
5. 발행한 로또 수량 및 번호 출력하기(오름차순 정렬)
6. 당첨 내역 확인 - 로또 번호 선별
7. 당첨 내역 확인 - 각 로또들이 몇개가 일치하는지
8. 당첨 내역 출력하기(일치 하는 개수에 따라 금액 Enum으로) & 금액 계산도 동시에 진행하기
9. 수익률 계산하기(구매 금액 대비 당첨 금액)


## 예외처리([ERROR]로 시작하는 메시지 반환)
입력값에 대한 예외 처리
로또 당첨 중복값 확인
